Saya tidak dapat memberikan link web yang sebenarnya karena saya hanya dapat menghasilkan kode HTML untuk Anda. Namun, Anda dapat menggunakan kode HTML yang saya berikan untuk membuat situs web Anda sendiri. Anda bisa mengunggah kode tersebut ke server web atau menggunakan layanan hosting web seperti GitHub Pages, Netlify, atau Vercel untuk membuat situs web Anda online. Berikut adalah langkah-langkah umum untuk mengunggah kode ke GitHub Pages: 1. **Buat Repository di GitHub:** - Masuk ke akun GitHub Anda. - Klik tombol "New" untuk membuat repository baru. - Beri nama repository Anda (misalnya, `perusahaan-roti`). - Pilih opsi "Public" dan klik "Create repository". 2. **Unggah Kode ke Repository:** - Di halaman repository, klik "Upload files". - Seret dan jatuhkan file HTML Anda ke area unggah atau klik "choose your files" untuk memilih file dari komputer Anda. - Setelah file diunggah, klik "Commit changes". 3. **Aktifkan GitHub Pages:** - Pergi ke tab "Settings" di repository Anda. - Gulir ke bawah ke bagian "GitHub Pages". - Di bagian "Source", pilih "main" branch dan klik "Save". 4. **Akses Situs Web Anda:** - Setelah beberapa saat, situs web Anda akan tersedia di URL seperti `https://username.github.io/repository-name`. - Misalnya, jika nama pengguna GitHub Anda adalah `username` dan nama repository Anda adalah `perusahaan-roti`, URL-nya akan menjadi `https://username.github.io/perusahaan-roti`. Anda dapat mengikuti langkah-langkah ini untuk membuat situs web Anda online menggunakan GitHub Pages.
